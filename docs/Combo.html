# <Combo />

This section represents the structure for a set of three moves, known as a combo. It contains 4 stance components and 3 move components.
Parameters:
Name Type Description
moveClick function Click event that sets target row and column.
moveDelete function Click event that removes current move data from a target row and column
row int The row number.
rowState Array.<string> An array of move names.
stances Array.<int> An array of stance positions.
